## 📖 简介（Introduction）

本模板旨在解决北航软件学院研究生在撰写 **文献综述（Literature Review）** 和 **开题报告（Proposal Report）** 时，格式与学校毕业论文模板不一致的问题。

`README_buaa.md` 是北京航空航天大学硕博学位论文 LaTeX 模板的 readme 原文。
